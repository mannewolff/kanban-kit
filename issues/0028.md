## Kontext
Die Board-Karten sind aktuell schlicht (Nummer + Titel + Mini-Kürzel-Badge). Die Toolbox (`../java/frontend/src/pages/kanban/KanbanCard.tsx`, `KanbanColumn.tsx`, `EpicBadge.tsx`, `cleanupCountdown.ts`) zeigt eine deutlich reichere Karte: Epic-Zeile + farbiger Rand, ⋮-Menü, Archiv-Countdown. Ziel: wortgetreuer Port, Enum→dynamische Board-Spalten. Für den Countdown braucht das Frontend die Retention-Tage aus dem Backend.

## Aufgabe
- `frontend/src/components/EpicBadge.tsx` an Toolbox angleichen: ●-Punkt + Kürzel auf getöntem `${hue}22`-Grund (statt aktuellem Voll-Chip).
- `frontend/src/components/BoardView.tsx` (Karten-Rendering nach `KanbanCard.tsx`):
  - Epic-Zeile (EpicBadge) oben + farbiger linker Rand (`borderLeft 4px` in Epic-Farbe) bei zugeordneter Karte.
  - "#num – **Titel**"; ⋮-Menü je Karte: Bearbeiten (öffnet Detail im Edit-Modus), Archivieren, "Nach <Spalte>" je andere `board.columns`; archiviert: Wiederherstellen / Endgültig löschen.
  - Archiv-Countdown "wird in X Tagen archiviert" auf Done-Karten (`movedToDoneAt != null`).
- `frontend/src/lib/cleanupCountdown.ts` (Port): `cleanupDaysRemaining`, `cleanupCountdownLabel`.
- `frontend/src/lib/statusColors.ts`: `ARCHIVED_STATUS_COLOR` ergänzen.
- `frontend/src/api/cards.ts`: `archive`/`restore`/`remove` ergänzen (Backend-Endpunkte POST /archive, /restore, DELETE existieren).
- `frontend/src/pages/BoardPage.tsx`: globaler "+ Neues Item"-Button im Board-Header (öffnet Anlege-Dialog, Default erste Spalte); Reload nach Archivieren/Wiederherstellen/Löschen/Move; `retentionDays` laden und an BoardView geben.
- Backend: neuer `GET /api/config` liefert `{ doneRetentionDays }` aus `CleanupProperties` (für den Countdown). Frontend `api/config.ts`. Backend-IT: Endpunkt liefert den konfigurierten Wert.
- Tests: EpicBadge-Restyle, cleanupCountdown (rein, fixed-date), ⋮-Menü Aktionen (Archivieren/Move rufen API), Countdown-Label auf Done.

## Akzeptanzkriterium
- `npm test` + `npm run build` grün; `mvn verify` grün (config-Endpoint-IT).
- Board-Karte zeigt Epic-Zeile + farbigen Rand, ⋮-Menü mit Bearbeiten/Archivieren/Verschieben, Done-Karten den Archiv-Countdown; "Neues Item"-Button im Header legt eine Karte an.

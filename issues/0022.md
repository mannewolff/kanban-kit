## Nachtrag: Bugfix (Commit ba45c46)

Beim Test gemeldet: Klick auf „Board" crasht die App (Konsole: 'Cannot read properties of undefined (reading length)').

Ursache: `useMatch('/boards/:id/*') ?? useMatch('/boards/:id')` in AppShell verletzte die Rules of Hooks — der `??`-Short-Circuit rief den zweiten useMatch nur bedingt auf, wodurch sich die Hook-Anzahl zwischen Nicht-Board- und Board-Route änderte.

Fix: ein einzelnes Splat-Muster `useMatch('/boards/:boardId/*')` matcht beide Routen. Regressionstest (Übergang / -> /boards/1) ergänzt.

Checks: npm test → 35 grün, npm run build → grün.
